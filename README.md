# HexSoftwares_gradecal
This is a simple Grade Calculator project developed using [java programming].
It helps users to calculate grades based on marks entered.
## Features
- Calculate total marks
- Calculate percentage
- Display grade (A, B, C, etc.)
- User-friendly interface
- ## Technologies Used
- Java / Python
- VS Code
## How to Run
1. Clone this repository
2. Open in VS Code
3. Run the program
## Output
Enter marks: 85
Grade: A
